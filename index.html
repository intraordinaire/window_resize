Parameters

Param Type Description Default
callback function The function you want to execute on window resize. null
timeout Integer (milliseconds) If you want a delay before sending the function. 0
type String ('width', 'height' or 'both') Listen the 'width', 'height' or 'both' resize. both
diff_height Integer or false If set, the minimum diff in pixels height to fire event, else fire event for each px. false
diff_width Integer or false If set, the minimum diff in pixels width to fire event, else fire event for each px. false

Example

jQuery(window).windowResize(
{
	callback: onWindowResize,
	timeout: 500
});
	
function onWindowResize(evt) {
    console.log('onWindowResize');
};
