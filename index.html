window_resize

Attach it to the window. It resolve the IE < 9 problem of window resize who fire event on each DOM element.

PLUS :

  • You can add a timeout.
  • You can chose if you want to watch only "width", "height" or "both" resize.
  • You can add a minimum "diff" pixels on width and height resize.
  • You can add steps to fire event only for certain sizes.

Example

    
jQuery(window).windowResize(
{
    callback: onWindowResize,
    timeout: 500
});
    
function onWindowResize(evt) {
    console.log('onWindowResize');
};
